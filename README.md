# DreamMoon Skills for ClawHub

这是 zfanmy-梦月儿 创建的 OpenClaw 技能集合。

## 技能列表

### 1. openclaw-deploy
**功能**：OpenClaw 打包部署工具

- Docker 镜像构建（clean/full 版本）
- 便携版打包
- 远程部署脚本

**安装**：
```bash
clawhub install openclaw-deploy
```

### 2. searxng-search
**功能**：SearXNG 网络搜索

- MCP 服务器集成
- Bash 搜索脚本
- 可配置 SearXNG 端点

**安装**：
```bash
clawhub install searxng-search
```
